# Interview Scheduler
A react application which allows users to book and cancel interviews 

# Technical Specifications
  React
  Webpack, Babel
  Axios, WebSockets
  Axios

## Setup

Install dependencies with `npm install`.

```sh
npm start
```

```sh
npm test
```
